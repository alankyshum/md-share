Initial public release. Extracted from the dotfiles-bundled `share--markdown` skill as a standalone, framework-agnostic markdown renderer with interactive tables (Tabulator), charts (Chart.js + mermaid pie/xychart auto-upgrade), gantt (frappe-gantt + mermaid auto-upgrade), mindmaps (markmap + mermaid auto-upgrade), maps (MapLibre + OpenRouteService), and fullscreen viewer.
