# md-share Public Toolkit

This repository is the public standalone successor to the legacy dotfiles-bundled `share--markdown` skill. It provides a publicly-installable, encrypted, and decentralized markdown sharing system backed by public GitHub repositories instead of centralized databases or Cloudflare KV stores. For detailed notes on architecture, transition boundaries, and phase-by-phase design, please refer to the [Implementation Plan](IMPLEMENTATION_PLAN.md).

Full docs land in Phase 5.
